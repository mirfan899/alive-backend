### Alive

### install packages

```shell
pip install -r requirements.txt
```


### upload image and video for vector db

```shell
curl -X POST http://127.0.0.1:5000/upload \
  -F 'image=@/path/to/your/image.jpg' \
  -F 'video=@/path/to/your/video.mp4'

```
